API Guide

You can access different data sets by the end points. Please find above for some examples for the endpoints for their respective data sets.

Here are some endpoints for the API

  • GET /api/topics - a list of topics
  • GET /api/users - a list of users
  • GET /api/articles - a list of articles
  • POST /api/articles - create an article
  • GET ~/:articleid - the article with the corresponding article ID
  • GET ~/:article/comments - GET the comments in the article
  • POST ~/:article/comments - create a comment to the article
