API Guide

You can access different data sets by the end points. Please find above for some examples for the endpoints for their respective data sets.

Here are some endpoints for the API

  • Get

    • /api/topics - a list of topics
    • /api/users - a list of users
      • /api/articles - a list of articles
      • ~/:articleid - the article with the corresponding article ID
      • ~/:article/comments - GET the comments in the article

  • Post

    • /api/articles - create an article
    • ~/:article/comments - create a comment to the article

  • Patch

    • /articles/:articleid - update article votes
    • /comments/:commentid - update comment votes

  • Delete

    • /comments/:commentid - delete comment votes
